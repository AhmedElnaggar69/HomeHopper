HomeHopper


## Features
- User and owner authentication
- Apartment search with location-based filtering
- Owner dashboard for listing management
- Chat system using WebSockets

## Setup
1. Clone the repository: `git clone https://github.com/yourusername/HomeHopper.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Set up a virtual environment: `python -m venv venv`
4. Create a `.env` file with `SECRET_KEY` and `DATABASE_URL`
5. Run migrations: `python manage.py migrate`
6. Start the server: `python manage.py runserver`
